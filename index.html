CrossAir

A Crosshair for FPS

Play Akimbo in FPS

This program is allowing to have a custom crosshair to play akimbo in all FPS games. You can change the crosshair before starting the program. It's just the file you tweak to replace under same folder of the program. Let's see...

Considerations About the Program

If you remove the PNG file, the GIF file will be used and crosshair animated. Press Keyboard keys page up to put in background and page down to put in foreground. I also made another program for having a background like rain, running on same principle, named CornFlake. To see it, your game, you are playing, must be in windowed mode. For games in windowed mode but still displaying a border at top, I made a tool to remove it. To note, the page up and page down keys are used to remove/add the border at top. Finally, I also made an television overlay named TeleVision, but you must use keyboard key page down under your game in foreground, not under the overlay, and it don't need WindowTitleRemover.
